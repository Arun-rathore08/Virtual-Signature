#Virtual-Signature

# Requirements

OpenCV

MediaPipe

Numpy

Streamlit

Collections

warnings

PIL

RepotLab

## how it works-

install all dependencies 

run the script- streamlit run Signature.py

Run air canvas 

draw your signature

Keyboard controls 

for colours:

          "r" for red
          
          "b" for blue
          
          "g" for green

          "y" for yellow

to save the signature press "s" and to quit the window press "q"
